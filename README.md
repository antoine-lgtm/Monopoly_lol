# Monopoly Frontend

Jeu de Monopoly multijoueur en ligne.

## Structure
- `index.html` — Page principale
- `js/` — Scripts du jeu
  - `lobby.js` — Gestion du lobby multijoueur
  - `multiplayer-client.js` — Client Socket.IO
  - `classicedition.js` — Données édition classique
  - `monopoly.js` — Logique principale du jeu
  - `ai.js` — Intelligence artificielle
- `css/` — Styles
- `images/` — Images du jeu

## Déploiement
Hébergé sur Netlify.
